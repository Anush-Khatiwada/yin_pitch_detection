YIN Pitch Detection Algorithm
Final Year Project - Mid Defense Implementation

Overview
A Python implementation of the YIN algorithm for fundamental frequency (F0) estimation in monophonic audio signals. Includes vectorized difference functions, cumulative mean normalization, and parabolic interpolation.

Features
Optimized vectorized CMNDF calculation
Audio downsampling for real-time speed
Dynamic thresholding to prevent octave errors
Live microphone input support
Dependencies
Python 3.x
NumPy
SciPy
Matplotlib
tqdm
